# Steps to Create Django RestAPI

  1. Install Django RestAPI
  2. Add Apps into Settings.py file
  3. Create Model (models.py)
  4. Create Serializers (serializers.py)
  5. Create Viewsets (views.py)
  6. Create Routers (urls.py)

Run Your Project.
